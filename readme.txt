== Description ==
A very simple bar for outgoing links in your posts.

<b>Features:</b>
<ul>
<li>Feature a "DiggBar like" or "Facebook like" to all external links in your posts</li>
<li>Whitelist for links that you do not want to apply the WP Super Bar</li>
<li>Share your links easly with facebook, digg, twitter, etc...</li>
<li>You can easly edit WP Super Bar CSS from your WordPress Backend</li>
<li>You can customise text and logo.</li>
</ul>

== Installation ==

1. Install and activate the plugin through the 'Plugins' menu in WordPress
2. Update the settings and design of your WP Super Bar going to Settings>WP Super Bar.

This plugin turn your links automaticly, to try just clic on outgoing link in your posts.
